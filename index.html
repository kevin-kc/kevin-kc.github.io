About:

  I'm Kevin Kwong-Chip, currently a fourth year computer science student at the University of Toronto set for graduation in May 2017 if everything goes as planned. One of the reasons I chose the field of software is in hopes that I get to create something that other people can use to better themselves, or make their lives/tasks more efficient.


About The Site:

  This website is meant to be a personal project in a public space where I can share my work projects and experiences. The site was also created with the intent of preventing myself becoming rusty with some programming languages. Having said that, this website was created from scratch using HTML5,CSS3, and JScript/JQuery with some additional libraries. As of writing this, the website is unfinished and besides the missing content, there are plans to adapt the site for mobile and add additional features. The site displays best for Chrome at the moment as that was the test browser, changes will be made eventually to add the support for FireFox. A time breakdown follows of how I went about building the site so far (time allocating is generous). The planning/design phase took approximately 15 hours total. That number includes the time spent thinking about the layout of the website as well as the look of the website's features ("front end", CSS). My main goal when designing the site was to have a simple, clean, intuitive look and also have everything be self contained within the site. Research was roughly 5 hours, comprising of mostly api searches on tools such as Bootstrap, Velocity and how to write the code so the behaviour of certain feature will be this or that. The HTML and JavaScript for the layout and content took around 15 hours thus far. This part was fairly straight forward but there were multiple revisits to the design phase once actually implementing the content.

  The black and off white contrast was done with intention to make looking at the site easy on the eyes. The lack of text on the main page was done for simplicity as to not make the user feel overwhelmed with options, and the vertical text and selection bars were done for a change in the typical design pattern you see on blogs or other sites showing off a portfolio. The header contains contact info, but I have purposefully hidden it away like that since likely if a user is visiting this site they would already have my contact info, as well as not wanting to clutter the selection space with text.
